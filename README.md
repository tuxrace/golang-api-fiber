# Golang API Fiber
> REST API using golang fiber with React on frontend

## Install
```bash
   npm install
   npm start
```
This will run both `front end` and `back end`

## API
| Method   |      API               |
|----------|------------------------|
| GET      |  /api/appliances`      |
| GET      |    /api/appliances/1   |
| POST     | /api/appliances        |
| DELETE   | /api/appliances/1      |
| GET      | /api/appliances-search?category=model&search=Tub |

## API Usage

### Add an appliances
